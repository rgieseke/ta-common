Command line switch to load a Lua script and exit.

 
module('_m.common.luaonly', package.seeall)
¶

Commands

 





  

    ¶
  

  

Run a script and exit Textadept.

Parameter:

file: Script to run.





  
local function luaonly(file)
  if not file then
    print('No script file given.')
    os.exit()
  end
